🥪 BLT-Sammich

A serverless Slack bot for the OWASP BLT community. Integrates with GitHub to track contributors, manage projects, and discover OWASP resources — all from Slack.

Slash Commands

/contributors

Displays contributor activity for the OWASP-BLT/Lettuce repository over the last 7 days.

/ghissue [title]

Creates a new GitHub issue in the configured repository directly from Slack.

/project [name]

Retrieves information about OWASP projects. Browse 800+ projects interactively.

/repo [technology]

Finds repositories based on technology stack or programming language.

API Endpoints

POST /slack/command
Receives Slack slash command payloads (HMAC-verified)
POST /slack/action
Receives Slack interactive component callbacks (HMAC-verified)
